# Electron Scaffolding

Empty electron project that provides a basic structure for native cross-platform desktop apps

![](http://drops.ricardoalcocer.com/drops/Screen%20Shot%202019-05-20%20at%202.02.24%20PM.png)

### Project structure

```
📁 assets
    📁 icons
        📁 darwin
            📄 icon.icns
        📁 png
            📄 16x16.png
        📁 win32
            📄 icon.ico
📁 bindings
    📄 mainwindow.js
📁 css
📁 fonts
📁 menus
    📄 mainwindow.js
📁 views
    📁 darwin
        📄 mainwindow.html
    📁 linux
        📄 mainwindow.html
    📁 win32
        📄 mainwindow.html
📄 .gitignore
📄 main.js
📄 package.json
📄 README.md
```
